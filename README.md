# Samayajna


```markdown


**Samayajna** is a React Native Expo app available for both Android and iOS. It helps you track and count the time you spend productively.

## Features

- **Track Productive Hours**: Start and stop a timer to measure how long you spend on productive tasks.
- **Daily/Weekly Insights**: View detailed analytics showing how productive you are over time.
- **Task Categorization**: Group your tasks into categories (e.g., Work, Study, Exercise).
- **Goal Setting**: Set daily or weekly productivity goals and track your progress.
- **Pomodoro Timer**: Use the Pomodoro technique with customizable work and break intervals.
- **Focus Mode**: Activate a distraction-free mode to enhance your concentration.
- **Cross-Platform**: The app is compatible with both Android and iOS devices.

## Installation

### Prerequisites

Before you begin, ensure you have the following installed on your machine:

- [Node.js](https://nodejs.org/)
- [Expo CLI](https://docs.expo.dev/get-started/installation/)

### Setup

1. Clone this repository:

   ```bash
   git clone https://github.com/YourUsername/Samayajna.git
   cd Samayajna
   ```

2. Install the required dependencies:

   ```bash
   npm install
   ```

3. Run the app on an Android or iOS emulator, or directly on your mobile device via Expo Go:

   - For Android:
     ```bash
     expo start --android
     ```
   
   - For iOS:
     ```bash
     expo start --ios
     ```

   Alternatively, scan the QR code with the Expo Go app on your phone.
